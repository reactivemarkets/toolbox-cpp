3 ## Swap
4 
5 Useful utility for insertion sort.
6 
7 ``` c
8 void node_swap(Node p, Node q)
9 {
10  Node prev = p->prev;
11  Node next = q->next;
12 
13  // First link.
14  prev->next = q;
15  q->prev = prev;
16 
17  // Second link.
18  q->next = p;
19  p->prev = q;
20 
21  // Third link.
22  p->next = next;
23  next->prev = p;
24 }

27 ## Sum Of Squares
28 
29 A numerically stable algorithm is given below. It also computes the mean. This algorithm is due to
30 Knuth, who cites Welford.
31 
32  sum_sq = 0
33  mean = x[1]
34  for i in 2 to n:
35  sweep = (i - 1.0) / i
36  delta = x[i] - mean
37  sum_sq += delta * delta * sweep
38  mean += delta / i
39  end
40 
41 The following algorithm (in pseudocode) will compute the covariate version with good numerical
42 stability.
43 
44  sum_x2 = 0
45  sum_y2 = 0
46  sum_xy = 0
47  mean_x = x[1]
48  mean_y = y[1]
49  for i in 2 to n:
50  sweep = (i - 1.0) / i
51  delta_x = x[i] - mean_x
52  delta_y = y[i] - mean_y
53  sum_x2 += delta_x * delta_x * sweep
54  sum_y2 += delta_y * delta_y * sweep
55  sum_xy += delta_x * delta_y * sweep
56  mean_x += delta_x / i
57  mean_y += delta_y / i
58  end
